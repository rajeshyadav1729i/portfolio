# Rajesh Yadav - Portfolio

## About
A modern, responsive portfolio website for Rajesh Yadav - Software Developer & Engineering Student.

## Features
- Modern UI with smooth animations
- Responsive design
- Dark/Light theme support
- Project showcase
- Skills and experience section
- Contact form

## Technologies Used
- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React Icons

## Getting Started

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```
